# 亚热带关卡游玩指南

欢迎来到亚热带丛林。在这里，你的目标是生存下去并最终找到通往文明世界的村庄。

## 1. 初始阶段：基础生存

当你开始游戏时，你通常处于**平原**。

* **首要任务**：收集基础资源。通过“探索”动作寻找**浆果**（补充食物和水分）和**木棍**。
* **寻找水源**：平原上的水通常是**脏水**。除非万不得已，否则不要直接饮用，这会大幅降低你的健康值。
* **管理精力**：不要连续执行高强度动作。利用“休息/庇护”动作恢复精力，尤其是在你拥有充足食物和水分时，休息还能缓慢恢复生命值。

## 2. 中期阶段：工具与火

为了更长久地生存，你需要进入**森林**。

* **制作工具**：收集石头、木头和草绳来制作**石斧**。有了斧头，你可以在森林中执行“砍伐树木”动作，获得大量木材。
* **生火**：利用收集到的引火物和木棍制作**手摇钻**。在地点搭建营火，这不仅能提供温暖，还是处理食物的关键。
* **烹饪**：将收集到的脏水在营火上烧开，获得安全的**开水**。将生肉（如兔肉、鱼肉）烤熟，以获得更高的营养价值并避免生食带来的健康风险。

## 3. 进阶阶段：狩猎与探索

随着装备的提升，你可以尝试获取更高级的资源。

* **捕鱼**：如果你到达了**河岸**，利用钓竿可以获得稳定的鱼肉来源。
* **狩猎**：在平原或森林，如果你拥有捕兽夹或猎枪，可以尝试狩猎兔子甚至熊。
* **寻找小屋**：在旅途中，如果你运气好发现了一座**废弃小屋**，一定要深入探索。那里可能藏有现代化的生存工具（如净水器、急救包）或强力武器。

## 4. 终极目标：寻找村庄

你的最终目标是不断推进**旅程进度**。

* **移动逻辑**：执行“移动”动作会消耗时间并推进路线进度。
* **环境适应**：注意昼夜交替。夜晚视野受限且可能更危险，建议在夜晚寻找安全地点（如洞穴或小屋）休息或进行室内合成。
* **胜利条件**：当你看到**村庄**场景时，执行“逃离荒野”动作即可赢得游戏。

## 生存小贴士

1. **负重管理**：不要收集所有东西。优先携带高价值的工具和高能量的食物。
2. **新鲜度优先**：先吃掉容易腐烂的浆果和生肉，将坚果和能量棒留作应急储备。
3. **保持干燥**：注意地点的湿度。潮湿不仅会影响你的状态，还会让你的引火物失效。
